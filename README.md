# Smart Tasks - Mobile Computing Lab

## Description
Smart Tasks is a task management mobile application built using React Native and Expo.

## Features
- Add tasks
- Edit tasks (long press)
- Delete tasks
- Categories
- Priority levels
- Due dates
- Filtering system
- Progress tracking
- Local storage using AsyncStorage

## Technologies
- React Native
- Expo
- JavaScript (JSX)
- AsyncStorage

## How to Run
```bash
npm install
npx expo start
